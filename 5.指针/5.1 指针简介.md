# 符号区分
&：取地址
*: 指针，保存变量地址 
->: 通过指针访问结构成员

# 指针表示
int *p; 定义指针变量
p = &i; 指针变量存放变量地址
int *p, q = int* p,q
const *p：指针是const，指针一旦指向某个变量，就不能再指向其他变量，但可以通过指针修改该变量的值
*const p：指针指向的值是const，不能通过指针修改该变量的值，但指针可以指向其他变量

# 指针运算
指针+1：加一个sizeof(指针所指向的变量的数据类型)
两个指针相减：之差除以sizeof
不同类型的指针不能互相运算

# 注意事项
变量的值是内存的地址， 普通变量的值是值本身，指针变量的值是具有实际值变量的地址
传入函数的是变量的地址，而非值本身，函数内通过指针访问变量
地址大小与int是否相同取决于编译器
必须对变量取地址
数组变量是特殊的指针（const指针）（函数参数表里int a[]和int *a等价）（*p = p[0]）
