
编译预处理指令:
- 以#开头，它不是C语言的成分，不要带；
#define用来定义一个宏
- 如果宏超过一行，需要用反斜杠\连接
- 宏后面可以有注释
- 宏可以带参数。带参数的宏的原则：整个值要有括号，参数出现的每个地方都要有括号

```
#include <stdio.h>

// const double PI = 3.14159;
#define PI 3.14159
#define cube(x) ((x)*(x)*(x))

int main(int argc, char const *argv[])
{
    int i;

    printf("%f\n", 2*PI*3.0);
    printf("%d\n", cube(3));
    printf("%d\n", cube(i+2));

    return 0;
}
```
